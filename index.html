Setup Guide

01

Install the app

Run the setup .exe and follow the installer. The package includes the desktop app, Electron runtime, icon assets, and the bundled runtime needed by manga extensions. Normal users do not need Node.js, npm, Java, or any additional tooling.

02

Add your first sources

watchAny does not ship pre-installed third-party sources. Open Settings > Sources and choose the tab for the content type you want to add.

03

Manga and manhwa

Under Sources > Manga, paste a Keiyoushi-compatible repository URL, click Validate & add, then install the extensions you want. Only install extensions from repositories you trust.

04

Movies, TV and anime streams

Under Sources > Media, paste any compatible media manifest URL for streams, torrents, catalogs, metadata, or subtitles. If the add-on requires setup, configure it first, then paste the configured manifest URL.

05

Start browsing

Catalog add-ons appear in the Movies catalog dropdown after they are enabled. Stream and torrent add-ons appear on detail pages when a matching title is opened. Use the Search tab to find anime, manga, or movies by name.

06

Optional cloud sync

The app works fully offline and locally. If you want your library and progress to sync across devices, enable account sync in Settings > Account. You can configure a custom sync server if needed.

Features

Anime and movies

Stream or download using torrents, media manifests, and configured sources. Built-in playback with subtitle support from multiple providers.

Manga and manhwa

Read with a full-featured reader that supports direction control, fit mode, zoom, border crop, background color, and continuous scrolling.

Local library

Track your progress across anime, manga, and movies with automatic chapter and episode tracking. Optional sync keeps your data safe across devices.

Customizable

Add third-party sources through Keiyoushi-compatible manga extensions and media add-ons. Configure torrent providers, subtitle services, and more.

Torrent streaming

Stream anime and movies directly from torrents without waiting for a full download. Supports DHT, PEX, and magnet links with real-time playback.

Downloads

Download anime episodes, movies, and manga chapters for offline viewing. Manage active downloads and track queue progress from the Downloads page.
